Inverse List
=====================================================================
 
## Problem description

#### function name
`inv_app`

#### input
2 lists
- `list1`
- `list2`

#### output
Inverse of list2 + inverse of list1

### Example
```
list1 = (1 2 3)
list2 = (a b c)
```
```
output = (c b a 3 2 1)
```
