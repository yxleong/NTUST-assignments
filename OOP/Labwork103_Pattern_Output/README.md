# Pattern Output

## Problem Statement
Write a program to display the following “*” pattern on the screen.

## Input
None.

## Output
Display the following “*” pattern on the screen.
```
       *
      *
     *
*   *
 * *
  *
```

* Eazy, only basic programming syntax and structure are required.
