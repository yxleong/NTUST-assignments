# List

## Problem statement 1
Write a program to add item 7000 after 6000 in the following Python List:
```
list1 = [10, 20, [300, 400, [5000, 6000], 500], 30, 40]
```

### Output
```
[10, 20, [300, 400, [5000, 6000, 7000], 500], 30, 40]
```
